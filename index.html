Featured · XAI 2024

XAI Cultural Heritage Prediction

A machine learning pipeline trained to predict cultural heritage site significance using multi-modal features — region, material composition, age, and structural data. SHAP values render the model's reasoning layer-by-layer, making predictions auditable and interpretable for domain experts and policy-makers.

  • SHAP-driven feature attribution per prediction
  • Multi-class classification with ensemble models
  • Interactive explanation visualizations

Full-Stack · AI Fairness 2024

Fairness AI Dashboard

A full-stack bias evaluation system built to audit trained models across demographic groups. Computes disparate impact ratios, equalized odds, and demographic parity scores. Stores audit logs in MongoDB and surfaces findings via an interactive dashboard.

  • Real-time bias metric computation
  • MongoDB-backed audit logging
  • Comparative group-level visualizations

ML · Healthcare 2023

Fetal Health Classification

A high-precision CatBoost classifier trained on cardiotocography data to detect fetal distress. Achieves ~97% accuracy through careful feature engineering, class balancing, and hyperparameter optimization. Validated with confusion matrix analysis across all three health classes.

  • ~97% accuracy with CatBoost
  • 3-class classification (Normal / Suspect / Pathological)
  • Full confusion matrix + precision/recall breakdown
