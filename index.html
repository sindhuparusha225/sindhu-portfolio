Featured XAI · Heritage

XAI-Based Cultural Heritage App Prediction

Trained an XGBoost classifier on cultural heritage data to predict app categories with 93.7% accuracy. The interesting part wasn't the accuracy — it was using SHAP and LIME to show exactly which cultural features drove each prediction. When a model makes a claim about cultural significance, it should be able to explain itself.
